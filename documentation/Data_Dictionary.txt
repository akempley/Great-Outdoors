| -------------------- | -------- | --------------------------------------------------------- | ------------------------ |

| `role`               | Text     | User role: `visitor`, `member`, `guide`, `staff`, `admin` | `member`                 |

| `profile_name`       | Text     | Display name on profile                                   | `John Nelson`            |

| `banner_image_url`   | URL      | Optional banner/header image                              | `https://.../banner.jpg` |
| `preferred_language` | Text     | UI language code, e.g. `en`, `es`                         | `en`                     |
| `created_at`         | DateTime | Date/time the account was created                         | `2025-02-10 18:30`       |
